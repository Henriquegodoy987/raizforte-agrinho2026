# 🌱 RaizForte

Projeto desenvolvido para o concurso **Agrinho 2026** na subcategoria 3 **Programação Front-End: HTML, CSS e JavaScript**.

O RaizForte é um site educativo e interativo criado para conscientizar sobre a importância da agricultura sustentável, mostrando como tecnologia, preservação ambiental e produção podem caminhar juntas para construir um futuro melhor.

---

# 🎯 Objetivo do Projeto

O projeto busca ensinar, de forma acessível e dinâmica, temas relacionados ao agronegócio sustentável, como:

- Produção agrícola
- Tecnologia no campo
- Sustentabilidade
- Solo e recursos naturais
- Colheita e distribuição
- Mercado de trabalho no agro

Além disso, o site possui um quiz interativo onde o usuário toma decisões que influenciam diferentes finais, incentivando reflexão sobre equilíbrio entre produção, tecnologia e preservação ambiental.

# 🌾 Temas Educativos

## 🌱 Produção Agrícola
Explica como funciona a produção moderna no campo, desde preparo do solo até logística e sustentabilidade.

## 🤖 Tecnologia no Campo
Mostra o uso de drones, sensores, automação, GPS agrícola e agricultura inteligente.

## 🌍 Sustentabilidade
Apresenta práticas sustentáveis que ajudam a preservar o meio ambiente e garantir o futuro da produção.

## 🪨 Solo e Recursos Naturais
Aborda conservação do solo, água, biodiversidade e equilíbrio ecológico.

## 🚚 Colheita e Distribuição
Explica o caminho dos alimentos do campo até a mesa do consumidor.

## 👨‍🌾 Trabalho no Agro
Mostra oportunidades profissionais modernas dentro do agronegócio.

---

# 🎮 Sistema do Quiz

O quiz funciona através de três atributos principais:

- 🌱 Sustentabilidade
- 📈 Produção
- 🤖 Tecnologia

Cada escolha altera os valores do jogador, levando a diferentes resultados finais.

Possíveis finais:

- 👑 Equilíbrio Perfeito
- 🌱 Guardião da Natureza
- 🤖 Mestre da Tecnologia
- 📈 Império da Produção
- 🏆 Inovador Verde
- 🏆 Magnata Tecnológico
- 🏆 Produtor Consciente
